
# Sistema de Licitaciones y Concursos Internos – Grupo 6

## 📌 Descripción
Sistema web de licitaciones y concursos internos que permite la gestión de usuarios, publicación de convocatorias, envío de propuestas, evaluación mediante rúbricas y generación de reportes.

Este proyecto ha sido desarrollado como parte de la asignatura **Estándares de Programación**.
